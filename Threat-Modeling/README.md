# Threat Modeling

This folder contains threat modeling activities used to identify security risks before software deployment.

Topics covered:

* PASTA Framework
* Threat Analysis
* Vulnerability Identification
* Attack Surface Analysis
* Risk Assessment
* Secure Software Development
* Security Controls
* Data Flow Analysis

Projects in this folder include:

* **PASTA-Threat-Model-Sneaker-App.md** – Threat model of a mobile marketplace application using the Process for Attack Simulation and Threat Analysis (PASTA) framework.

These activities demonstrate my ability to analyze applications, identify threats and vulnerabilities, evaluate attack paths, and recommend security controls to reduce organizational risk.
