# PASTA Threat Model – Sneaker Marketplace Application

## Overview

Conducted a threat modeling exercise using the Process for Attack Simulation and Threat Analysis (PASTA) framework for a mobile sneaker marketplace application.

## Objectives

* Analyze business and security requirements
* Evaluate application technologies and architecture
* Identify threats and vulnerabilities
* Assess attack vectors and attack paths
* Recommend security controls to reduce risk

## Skills Demonstrated

* Threat Modeling
* Risk Analysis
* Security Assessment
* Application Security
* Vulnerability Identification
* Attack Surface Analysis
* Security Controls Evaluation
* Data Flow Analysis

## Framework Used

* PASTA (Process for Attack Simulation and Threat Analysis)

## Technologies Evaluated

* API
* Public Key Infrastructure (PKI)
* SHA-256
* SQL Database Systems

## Outcome

Identified potential threats, application vulnerabilities, and attack vectors affecting user data and transactions. Recommended multiple security controls including multi-factor authentication, encryption, least privilege access, and secure coding practices to strengthen application security.
