# Home Network Asset Inventory and Classification

## Project Overview

This project involved creating an inventory of assets within a home network environment and classifying them based on their value, sensitivity, and security requirements. Asset management is a fundamental cybersecurity practice that helps organizations identify what needs protection and prioritize security controls accordingly.

## Objective

The objective of this project was to:

* Identify devices and assets connected to a home network
* Classify assets according to their importance and sensitivity
* Assess potential security risks associated with each asset
* Practice asset management and documentation techniques used in cybersecurity

## Asset Inventory and Classification

| Asset                  | Type                   | Classification  | Reason                                                          |
| ---------------------- | ---------------------- | --------------- | --------------------------------------------------------------- |
| Personal Laptop        | Endpoint Device        | High Priority   | Stores personal files, accounts, and sensitive information      |
| Smartphone             | Mobile Device          | High Priority   | Contains personal data, authentication apps, and communications |
| Wi-Fi Router           | Network Infrastructure | Critical        | Provides network connectivity and controls network access       |
| Smart TV               | IoT Device             | Medium Priority | Connected to the network but stores limited sensitive data      |
| Cloud Storage Accounts | Data Asset             | Critical        | Contains important documents and personal information           |
| Email Account          | Information Asset      | Critical        | Used for communication, account recovery, and authentication    |
| External Hard Drive    | Storage Device         | High Priority   | Stores backups and important files                              |

## Risk Considerations

Several risks were identified during the asset classification process:

* Unauthorized access to network devices
* Weak passwords and authentication practices
* Unpatched software vulnerabilities
* Data loss due to hardware failure
* Exposure of sensitive information through compromised accounts

## Security Recommendations

* Enable multi-factor authentication (MFA) where possible
* Use strong and unique passwords for all devices and accounts
* Regularly update device firmware and software
* Perform routine data backups
* Secure the home wireless network using WPA2 or WPA3 encryption
* Monitor connected devices and remove unused assets
