# Security Controls

This folder contains security control, access management, and identity management activities based on cybersecurity frameworks and industry best practices.

Topics covered:

* Principle of Least Privilege (PoLP)
* Access Control
* Authentication and Authorization
* Identity and Access Management (IAM)
* Multi-Factor Authentication (MFA)
* NIST SP 800-53 Security Controls
* Data Leak Prevention
* Event Log Analysis
* Security Recommendations and Risk Reduction

Projects in this folder include:

* **Least-Privilege-Data-Leak-Analysis.md** – Analysis of a data leak scenario caused by excessive permissions and recommendations for implementing least privilege.
* **Access-Control-Incident-Investigation.md** – Investigation of a suspicious payroll transaction involving access control weaknesses, with recommendations to improve authentication, authorization, and account management.

These activities demonstrate my understanding of security controls used to protect organizational systems, data, and user accounts while reducing security risks and unauthorized access.
