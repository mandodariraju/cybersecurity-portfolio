# Security Controls

This folder contains security control, access management, identity management, and security awareness activities based on cybersecurity frameworks and industry best practices.

## Topics Covered

* Principle of Least Privilege (PoLP)
* Access Control
* Authentication and Authorization
* Identity and Access Management (IAM)
* Multi-Factor Authentication (MFA)
* NIST SP 800-53 Security Controls
* Data Leak Prevention
* Event Log Analysis
* Security Recommendations and Risk Reduction
* Social Engineering Awareness
* USB Security Risks and Attack Vectors

## Projects in this Folder

* **Least-Privilege-Data-Leak-Analysis.md** – Analysis of a data leak scenario caused by excessive permissions and recommendations for implementing least privilege.

* **Access-Control-Incident-Investigation.md** – Investigation of a suspicious payroll transaction involving access control weaknesses, with recommendations to improve authentication, authorization, and account management.

* **USB-Attack-Vector-Analysis.md** – Assessment of a USB baiting scenario, including analysis of sensitive information exposure, attacker techniques, potential organizational risks, and recommended security controls to mitigate removable media threats.
