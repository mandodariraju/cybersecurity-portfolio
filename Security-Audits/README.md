# Security Audit

This folder contains security audit and compliance assessment projects that demonstrate my ability to evaluate security controls, identify risks, assess compliance requirements, and recommend security improvements.

## Projects

### Botium Toys Security Audit

Conducted an internal security audit of Botium Toys using the NIST Cybersecurity Framework (CSF).

Key activities included:

* Reviewing existing security controls
* Identifying security gaps and risks
* Assessing compliance requirements
* Evaluating access control practices
* Reviewing data protection measures
* Recommending remediation strategies
