# Linux Labs

This folder contains hands-on Linux administration and cybersecurity labs completed as part of my cybersecurity learning journey. These labs demonstrate practical experience with Linux commands, system administration, file security, package management, and file integrity verification.

## Projects

### File Permissions in Linux

Reviewed and modified file and directory permissions using Linux commands such as `ls -la` and `chmod`. This activity focused on implementing the principle of least privilege and securing access to files and directories.

### Package Management with APT

Installed, removed, and verified security tools including Suricata and tcpdump using the Advanced Package Tool (APT). Demonstrated software management and system administration skills in a Linux environment.

### File Integrity Verification Using SHA-256

Generated and compared SHA-256 hash values using Linux tools such as `sha256sum` and `cmp` to verify file integrity and detect file modifications.
